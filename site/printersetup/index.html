    Granting Printer Access to New Employees¶

    1. Access the Printer Directory for the 5th floor of the MERB:

    2. Click the Start menu, type Run, and hit Enter.

    3. Type \\IU-OTA-PRNT01 and click OK.

    4. In the window that appears, use the Search in IU-OTA-PRNT01 box:

    5. Type 191225-HA5003 and press Enter.

    6. Select the printer when it appears.

    7. Right-click the printer and select Open.

    8. Go to the Printer tab and click Set as Default Printer.

    Note

    If the employee does not have access, contact htshelp@iu.edu for assistance.
